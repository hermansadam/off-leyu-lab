# off-leyu-lab

本仓库用于归档和发布多个独立 HTML 页面。这些页面不针对任何特定域名或网站，仅作为通用 HTML 资源的存储与展示。

## 项目简介

off-leyu-lab 是一个静态页面归档仓库，主要存放独立编写的 HTML 文件。每个页面均为自包含的 HTML 文档，可用于展示、测试、参考或存档目的。

## 目录说明

- `index.html` — 仓库入口页面（如有）
- `pages/` — 存放主要 HTML 页面文件
- `assets/` — 页面所需的静态资源（如 CSS、JS、图片等）
- `archive/` — 历史版本或不再维护的页面归档

> 目录结构可能随仓库内容更新而调整，请以实际文件为准。

## 页面归档说明

- 所有页面均为独立 HTML 文件，不依赖外部特定服务或数据库。
- 页面内容可能包含示例、实验、笔记或临时项目，不构成任何正式发布或承诺。
- 归档页面仅供学习、参考或内部使用，不保证长期可用性或兼容性。

## 维护说明

- 本仓库不定期更新，可能添加新页面或清理旧文件。
- 不提供 Issue 或 Pull Request 的响应保证，但欢迎合理建议。
- 仓库维护者保留对内容和结构的最终解释权。

## 使用建议

- 可直接通过浏览器打开 HTML 文件查看效果。
- 如需部署，建议将整个仓库克隆或下载后使用静态服务器托管。
- 所有资源均来自本仓库内部，无需额外下载。
